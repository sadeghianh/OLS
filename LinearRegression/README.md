### `ols_assignment.py`
OLS (Ordinary Least Squares) regression solved two ways:
- **Route A**: Slope via covariance over variance (sums table)

- **Verification**: numpy.polyfit + sklearn LinearRegression

Data points: (1,2), (2,3), (3,5), (4,6)

Result: ŷ = 0.5 + 1.4x
